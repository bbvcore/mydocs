# Tipos estructuras de datos
# Arrays - Mutables
```python
array = [10,20,30, "true", True]
array.append("final")
print("Array:", array)
```
# Tuplas - Inmutables
```python
tupla = ("CSS",20,"JS", False)
print("Tupla:", tupla[2])
```
